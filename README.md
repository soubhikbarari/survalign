# Illusion of Consensus
## Replication Code
### Author: Soubhik Barari

Directory of replication code, data, figures, and paper for 
"illusion of consensus" project.

## Running
- Prep all datasets: `source("prep/run_all.R")` (or `prep.R` wrapper).
- Run all analyses: `source("analysis/run_all.R")` (or `analyze.R` wrapper).
- Dataset-specific scripts live in `prep/` and `analysis/`.
